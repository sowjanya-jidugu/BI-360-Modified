# Power BI 360 Sales Analytics Dashboard (Modified)

This project is a modified version of the BI-360 Power BI dashboard created for sales analysis.

## Objective
To analyze company-wide sales performance and provide meaningful business insights.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key Insights
- Revenue and profit trends
- Region-wise sales analysis
- Product-wise performance
- Dynamic slicers and filters
